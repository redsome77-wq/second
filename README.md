# HTML
## HTML 구조태그
* HTML은 **웹문서**를 뜻한다.
* `태그` 웹문서에 들어가는 내용~제목 모든 요소는 `<태그>`로 묶여서 만들어진다.
## HTML 주의사항
* 파일명.html
* 모든 파일명, 폴더명은 영문 소문자 사용, 특수문자 금지, 공백 금지, 한글 금지, 숫자 앞 금지
* start.html = O
* start1.html = O
* 1start.html = X (숫자앞)
* basic start.html = X (공백)
* basic_start.html = O (허용 특수문자 _ -)
* basic%start.html = X (허용 외 특수문자)
* 시작.html = X (한글)

## 태그속성 class, id
### class
* `<태그 class="값">``</태그>`
* 2번 이상 반복하는 이름인 경우
* 예: lnb, snb, container 등
### id
* `<태그 id="값">``</태그>`
* 반복하면 안되는 단독 명칭인 경우
* 예: header, gnb, footer 등
## HTML 기초 문서 태그
* 블록태그 : w100%, 새로운 행, 크기가 큼
* 인라인태그 : 열(옆으로 나열), 크기는 내용만큼 인식.
### 제목
* `h1~h6` : 블록태그, h1대제목, h6소제목, h레벨이 높을수록(1에 가까울수록. 1~3 주로 사용) 검색어로 인식될 확률이 높음. SEO마케팅 목적.
### 내용
* `p` : 블록, 한줄~여러줄 단락형태의 내용일때
* `br` : 인라인, 줄바꿈할때 사용
* `em` : 인라인, 단순 문맥 강조, 주로 p 자식으로 사용, h에 사용 지양, **주로 내용에 사용**
* `strong` : 인라인, 중요+심각성 강조, p 자식 위주, h에 사용 지양
* `blockquote` : 블록, 긴 인용문
* `q` : 인라인, 짧은 인용문
* `code` : 인라인, 컴퓨터 명령어를 화면에 표시할때
* `address` : 블록, **footer**안 주소 및 연락처, 해당 서비스 주소만 포함, address안 블록 사용 주의 (예시: `p`는 블록이므로 adress안에서 못 씀)
### 레이아웃
* `div` : 블록, 그룹, 자식이 2개 이상이며 같은 방향일때, 블록과 인라인을 모두 가질 수 있음.
* `span` : 인라인, 그룹(위 div와 같음), **보조(디자인)**, 인라인만 자식으로 가질 수 있음.
* `hr` : 블록, 경계 구분용 가로선 (디자인 테두리 용도 아님)
### 특수문자 태그
* `&copy;` : footer 저작권 마크 ⓒ
* `&lt;` : <
* `&gt;` : >
* `&amp;` : &

### 링크태그 href 주소 작성법
* 절대경로 : `href="https://주소"` 실제 서비스되는 웹주소
* 상대경로 : `href="./폴더 및 파일"` 같은 위치 또는 같은?위치?
* 상대경로 : `href="../폴더 및 파일"` 상위 위치
### 바로가기 링크(스크롤 이동하여 특정 위치로 가기)
1. `<a></a>` 태그 준비
2. 1번 링크 클릭 시 이동하고 싶은 위치에 해당하는 태그에 `id="위치명"`입력하기, 예) `<body id="top"></body>`
3. 1번 링크에 2번 아이디 연결하기 `<a href="#이동아이디명"></a>` **('#'표기 필수)**

### 이미지 필수 속성
* `src` : 이미지 경로(**상대**/절대경로)
* `alt` : 대체 텍스트(의미가 있는 경우 `alt="의미"`. 이미지 설명문)
### 이미지 선택 속성
* `class`, `id`(이미지 외 모든 태그 동일)
## background-images (CSS 배경 이미지)
### `img`태그가 아닌 background-images 처리하는 경우
* 형제, 자식, 자손 요소와 겹쳐져서 디자인 된 경우
* 쇼핑몰 상품 썸네일, 히어로 배너, 광고 배너 등...
#### 태그에 배경 이미지 적용하는 작성순서
1. `<태그 style="">`
2. `<태그 style="background-images:;">`
3. `<태그 style="background-images:url(경로);">`
